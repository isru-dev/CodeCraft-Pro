## README: CodeCraft Pro Landing Page

A minimalist, high-end landing page for **CodeCraft Pro**, which is  educational platform .

### 🚀 Key Features

*   **Responsive Header**: A sticky navigation bar with a functional hamburger menu for mobile devices.
*   **Hero Section**: A "stunning" introduction featuring a typed-effect hero section and modern aesthetics.
*   **Course Grid**: Vertically long, premium product cards showcasing available programs like **Fullstack AI Development** and **Power BI**.
*   **Interactive Testimonials**: A horizontally scrollable success story section with "snap-to-center" behavior and manual navigation arrows.
*   **Smooth Navigation**: Internal page linking with smooth-scroll behavior and optimized scroll-margins for fixed headers.

---

### 🛠️ Tech Stack

*   **Frontend**: React (Vite)
*   **Styling**: Modern CSS (Flexbox, Grid, Glassmorphism)
*   **Deployment**: Vercel

### ⚙️ Setup Instructions

1.  **Clone the repository**:
    ```bash
    git clone <your-repo-url>
    ```
2.  **Navigate to the project root**:
    ```bash
    cd "CodeCraft Pro"
    ```
3.  **Install dependencies**:
    
```bash
    npm install
    ```
4.  **Run in development mode**:
    ```bash
    npm run dev
    ```
5.  **Build for production**:
    ```bash
    npm run build
    ```
